偵測到 file:// 協議 — API 無法直接呼叫
瀏覽器的安全政策會封鎖從本機檔案直接呼叫外部 API（CORS 限制）。
解決方法：用本機伺服器開啟此頁面：

1. 在檔案目錄開啟 start.bat
2. 瀏覽器自動開啟 http://localhost:8080
3. 再輸入 API Key 即可正常使用
